MeshSet Extender v1.0.1.3
### General
- MeshSet Exporter
  - "Export Mesh Res/Chunk Files" context menu item for Mesh assets.
  - "Export Res/Chunk Files" and "Export Textures" buttons in the mesh editor toolbar.
  - Export textures shows a file type dialog before exporting.
### Auto LOD
- Single and Batch Auto LOD import workflows.
  - Right-click folder → "Batch Auto LOD Import" / "Batch Auto LOD Import + Subfolders" (adds folder-scoped batch import to Data Explorer).
- Import settings
  - Debug logging toggle available in import settings.
- Logging / diagnostics
  - Completion summary includes mesh type tags: (Rigid), (Composite), or (Skinned).
  - Logs which preset and ratios were used for decimation.
  - Warns in logs if any section decimates to fewer than 10 triangles and suggests Conservative preset or higher ratios.
  - More verbose debug stack traces when DebugLogging enabled.
  - Auto-match by filename for bulk imports (matches mesh name to FBX filename).
## Fixed
### Auto LOD
- LODS not updating shaderblocks after LOD0 when importing a single mesh.
### Cloth Generator
- Tool tips showing wrong values.
- Improved error messages and logging for failed cloth extraction/adaptation steps.
## Requirements
- Frosty Editor 1.0.6.3
## Notes / Known Limitations
- Extremely small meshes may not simplify further (Auto LOD).
- Vertex collapsing is not used — index-only simplification approach for safety (Auto LOD).
- Some composite meshes may still require manual intervention.
